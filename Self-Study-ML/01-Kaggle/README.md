## 캐글 저장소

- 대부분의 data 파일은 업로드 용량 초과로 인하여 **별도 업로드 하지 않습니다.**
- 대부분의 캐글 file들이 initial commit 된 파일 상태로 남아 있습니다.
- `initial commit` 파일은 깔끔하게 코드 정리가 안 된 경우가 많으므로, 참고용 으로 봐주세요.
